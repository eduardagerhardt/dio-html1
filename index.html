Como surgiu a Internet? ↩︎

Na década de 1958, foi desenvolvida uma rede para fazer trocas de informações para as pesquisas militares dos EUA. O nomeando de Advanced Research Projects Agency.

Com o contexto de corrida espacial, em meio a Guerra Fria, os Soviéticos lançaram o Sputnik - o primeiro satélite artificial - para o espaço. Com esse evento o governo norte-americano desenvolveu o projeto que criaria a Internet.

A agência desenvolveu um sistema que permitisse envio de informações e dados importantes de pesquisa para proteção e defesa do país em caso de guerra nuclear. Essas informações eram enviadas para o Pentágono e centros militares de pesquisa. O projeto foi conduzido pelo Robert Taylor e Lawrence Roberts contando com o estudo de outros vários cientistas.

O projeto realizado pela Arpa foi renomeado de Apanet e inaugurada em 1969, na época, apenas quatro computadores nos EUA eram capazes de suportar o envio de dados dessa rede. Foi realizado um teste inicial com o envio da mensagem "log win", mas o computador de Stanford não suportou e travou com recebeu a letra "g". No segundo teste, momentos depois, teve sucesso e a partir daí uma conexão se estabeleceu permanente entre a Universidade da Califórnia em Los Angeles e SRI.

Já na década de 80 a Internet se expandiu seu uso não apenas para informações de defesa militar, mas, para fins comerciais e privados. 8 anos depois a rede chegou ao Brasil e sendo liberado para uso em 1994.

Hoje, a Internet é o principal meio de comunicação do planeta, com mais de 5 bilhões de usuários.

O que é HTML?↩︎

O Hypertext Markup Language é uma linguagem de marcação de hipertexto utilizada na criação de páginas web. Os marcadores são chamados de "tags". Essas tags servem para indicar uma função na página, como: textos, imagens, vídeos, tabelas...

Ela foi criada em 1990 pelo físico britânico Tim Berners-lee, para auxiliar na formação de documentos de pesquisa compartilhados entre ele e seus colegas.

Com a popularização da Internet, o HTML passou a ser utilizado na criação de sites, se tornando um dos pilares da "World Wide Web". A tecnologia atualmente evoluiu e com ele é utilizado o conjunto de outras ferramentas como o CSS e o JavaScript.

Estrutura básica do HTML ↩︎

!Doctype html ↩︎

Para organizar as informações que é recebida e interpretada para um navegador precisa começar a colocar o !Doctype html. Aqui nós temos um exemplo:

     

        <!doctype html>
        <html lang="pt-br">
        <head>
          <meta charset="utf-8">
          <meta name="viewport" content="width=device-width, initial-scale=1.0">
          <title>Exemplo HTML</title>
        <head>     
        <body>
        <p> Olá, mundo! </p>
        </body>
      </html>

O que é head? ↩︎

O header age como um recipiente de tudo o que você deseja incluir na página HTML. Não sendo o conteúdo e sim o cabeçalho. Onde terá descrições que queira que apareça nos resultados de busca. Podendo usar até o CSS para estilizar o conteúdo em sí.

O que é Body? ↩︎

O elemento body contém todo o conteúdo que queira mostrar para o público que visita a sua página, sejam eles textos, imagens, vídeos ou qualquer outra coisa.

O que é o hr? ↩︎

O <hr> nada mais é que uma linha. Na imagem à cima temos um exemplo de como fica dentro do navegador.

Porém, esse elemento tem uma característica. Ele não tem o código fechado.

Exemplo:

 
     

        <img src="www.imagem.com">
        <hr>

Estilos de marcadores ↩︎

Marcadores de textos

Texto - Parágrafo ↩︎

Para definir um parágrafo é preciso colocar <p> para fazer a linha de código. Definindo um espaçamento de linha

     

        <p> Olá, mundo! </p>

Texto - Títulos ↩︎

No HTML usamos o código <h> para colocar algum título. Porém, a cada H que é colocado o título vai diminuindo de tamanho

Como título,subtítulo, lide... assim por diante terminando no h6. Veja os exemplos abaixo:

     

        <h1> Olá, mundo! </h1>
        <h2> Olá, mundo! </h2>
        <h3> Olá, mundo! </h3>
                ...
        <h6> Olá, mundo! </h6>

Sendo assim:

 
    

    
Olá, mundo!
 
    
Olá, mundo

    
Olá, mundo

            ...
    
Olá, mundo

Texto - Negrito ↩︎

Para deixar o texto em negrito usamos o código <strong>

     

        <strong> Olá, mundo! </strong>

Texto - Itálico ↩︎

Para deixar o texto itálico basta usar o código <i>

     

        <i> Olá, mundo! </i>

Texto - Sublinhado ↩︎

Para deixar o texto sublinhado usamos o código <u>. U significa underline.

     

        <u> Olá, mundo! </u>

Texto - Blockquote ↩︎

O elemento <blockquote> é utilizado para criar citações em bloco. Ele é frequentemente usado para destacar citações de outras fontes ou para enfatizar uma parte específica do texto.

Veja o exemplo:

     

        <blockquote>
            <p> Esta é uma citação em bloco. </p>
        </blockquote>

Com a linha de código pronta:

     

        

            
Esta é uma citação em bloco.

O elemento blockquote geralmente é estilizado com recuo e formatação especial para diferenciá-lo do restante do texto. Ele pode conter outros elementos HTML, como parágrafos, listas e imagens.

Texto - Sup e Sub ↩︎

O elemento <sup> é utilizado para criar texto sobrescrito, enquanto o elemento <sub> é utilizado para criar texto subscrito.

Veja o exemplo:

     

        <p> O valor de x² + y² = z² </p>
        <p> H₂O </p>

Com a linha de código pronta:

     

        
O valor de x² + y² = z²

        
H₂O

Âncoras ↩︎

As âncoras são utilizadas para criar links dentro do HTML. Para isso usamos o código <a>. O elemento href é o que define o link. As âncoras são conhecidas como "texto diferente ou azul" dentro da página.

A URL de destino pode ser tanto externa quanto interna dentro da página. Até mesmo uma referência para um website interno. Ele pode se referenciar tanto HTML, gráficos, vídeos, imagens.

Para uma referencia de dentro do website precisamos fazer um id utilizando o carácter #. Ao clicar na palavra que está linkada a página "rola" até a região mencionada.

Um exemplo de links externos:

     

        <a href="https://www.google.com"> Google </a>

Um exemplo de link interno:

     

        <a href="index.html"> Página HTML </a>

Temos outros exemplos de links:

     

        <a href="https://www.google.com"> Google </a>
        <a href="index.html"> Página HTML </a>
        <a href="#ancoras"> Âncoras </a>

Com a linha de código pronta:

     

        Google
        Página HTML
        Âncoras

Ao clicar no link "Âncoras" a página irá rolar até a seção de âncoras.

Para criar um link que abra em uma nova aba, basta adicionar o atributo target="_blank" na tag <a>. Veja o exemplo:

     

        <a href="https://www.google.com" target="_blank"> Google </a>

Com a linha de código pronta:

     

        Google

Ao clicar no link, o Google será aberto em uma nova aba do navegador.

Além disso, você pode adicionar um título ao link usando o atributo title. O título aparecerá como uma dica quando passar o cursor sobre o link. Veja o exemplo:

     

        <a href="https://www.google.com" title="Acesse o Google"> Google </a>

Com a linha de código pronta:

     

        Google

Agora, quando passar o mouse sobre o link, verá a dica "Acesse o Google".

As âncoras são uma parte fundamental da navegação na web e permitem que os usuários se movam facilmente entre diferentes partes de uma página ou entre páginas diferentes.


Além disso, as âncoras também podem ser usadas para criar links para seções específicas dentro de uma mesma página. Para fazer isso, você pode usar o atributo id em um elemento HTML e, em seguida, criar um link para esse ID usando o caractere de hashtag (#) seguido pelo nome do ID.


Veja o exemplo:

     

        <p id="ancora1"> Esta é a âncora 1 </p>
        <a href="#ancora1"> Ir para a âncora 1 </a>

Com a linha de código pronta:

     

        
Esta é a âncora 1

        Ir para a âncora 1

Quando o usuário clicar no link "Ir para a âncora 1", a página rolará automaticamente até a seção marcada com o ID "ancora1".

As âncoras são uma parte essencial da navegação na web e permitem que os usuários se movam facilmente entre diferentes partes de uma página ou entre páginas diferentes. Elas são amplamente utilizadas em sites, blogs e aplicativos da web para melhorar a experiência do usuário.

Além disso, as âncoras também podem ser estilizadas usando CSS para se destacarem visualmente. Você pode alterar a cor, o tamanho e outros estilos do texto do link para torná-lo mais atraente.

Por exemplo:

     

        <a href="https://www.google.com" style="color: blue; font-size: 18px;"> Google </a>

Com a linha de código pronta:

     

        Google

Agora, o link "Google" terá uma cor azul e um tamanho de fonte maior.

Marcadores de Mídias ↩︎

Imagem e vídeo

Como mencionado no body, podemos incluir qualquer coisa nele. Veja o exemplo abaixo de como colocar uma imagem.

Exemplos:

Para colocar uma imagem/vídeo nós usamos o elemento <img>. Dentro dessa tag existe o elemento src que nada mais é que o source.

     

        <img src="www.imagem.com">

Ao colocar essa tag automáticamente a imagem irá vim com o tamanho real dela. Para selecionar o tamanho só basta colocar o código widht. Que significa largura.

Marcador de estruturas - ol e ul

Para criar uma lista ordenada usamos o elemento <ol>. Já para criar uma lista não ordenada usamos o elemento <ul>.

Dentro de cada um deles usamos o elemento <li> para definir os itens da lista.

Veja o exemplo abaixo:

Lista ordenada ↩︎

     

        <ol>
            <li> Item 1 </li>
            <li> Item 2 </li>
            <li> Item 3 </li>
        </ol>

Lista não ordenada ↩︎

Marcador de estruturas - Input Typer ↩︎

O elemento <input> é utilizado para criar campos de entrada de dados em formulários. O atributo type define o tipo de campo que será exibido.

Os tipos de campos mais comuns são:

     

            <input type="text" name="nome" placeholder="Digite seu nome">
            <input type="password" name="senha" placeholder="Digite sua senha">
            <input type="color" name="color" placeholder="Escolha sua cor">
  • text: campo de texto simples.
  • password: campo de senha, onde os caracteres digitados são ocultos.
  • email: campo para inserir endereços de e-mail.
  • number: campo para inserir números.
  • radio: botão de opção.
  • submit: botão para enviar o formulário.
  • file: campo para upload de arquivos.
  • date: campo para selecionar uma data.
  • color: campo para selecionar uma cor.
  • tel: campo para inserir números de telefone.
  • url: campo para inserir URLs.
  • search: campo para pesquisa.

    • Exemplos de input typer:

             
    
                <input type="text" name="nome" placeholder="Digite seu nome">
                <input type="password" name="senha" placeholder="Digite sua senha">
                <input type="color" name="color" placeholder="Escolha sua cor">

    Esses são apenas alguns exemplos de tipos de campos que podem ser criados com o elemento <input>. Existem muitos outros tipos disponíveis, cada um com suas próprias características e funcionalidades.
